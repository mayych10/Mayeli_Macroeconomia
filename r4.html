Capítulo 4: El modelo Macroeconómico IS-LM#

Parte 1: Reporte Lectura#

El texto “Demanda y oferta agregada en presencia de políticas monetarias no convencionales” presentado por Mendoza (2015) plantea como pregunta ¿De qué manera modelar la política monetaria no convencional? Para responder esta cuestión propone extender el modelo estándar de demanda y oferta agregada de economía cerrada para incorporar las innovaciones de la política monetaria aplicadas con el objetivo de contrarrestar las secuelas de la crisis internacional de 2008-2009. De manera que para analizar el impacto de la flexibilización cuantitativa añade un mercado de bonos a largo plazo (BB).

En cuanto a las fortalezas del documento, presenta el contexto que conllevo al cambio de las políticas por lo que permite al lector comprender la extensión del modelo de política monetaria. Además que incluye ejercicios de estática comparativa que muestra los efectos de las políticas macroeconómicas a corto plazo, en el tránsito al equilibrio, en el equilibrio estacionario y los efectos con las expectativas racionales, siendo este último una de sus principales fortalezas pues es una situación importante a tomar en cuenta al aplicar la política macroeconómica en la actualidad, donde el público cuenta con mayor información para predecir lo que sucede condicionando la política macroeconómica a esos pronósticos. Asimismo, otra fortaleza es que los gráficos cuentan con una detallada explicación de sus componentes.

Entonces, la contribución de este artículo es que el autor presenta la sección de efectos de las políticas macroeconómicas y los choques de oferta y los efectos dinámicos de las políticas macroeconómicas y los choques de oferta en los diversos escenarios propuestos. De estos se puede comprender lo que ocurre con las variables en los diversos escenarios, comprender los efectos sobre las variables endógenas del modelo y evaluar numéricamente las proposiciones formuladas.

Por otro lado, en cuanto a las debilidades se puede considerar que el artículo podría ofrecer una explicación más dinámica para componentes relacionados a las políticas macroeconómicas y los choques de oferta. Asimismo el limitarse a demostrar que los viejos modelos y métodos son todavía útiles sin compararlos con algunos más contemporáneos que puedan dar mayor validez a su posición de continuar usando estos modelo teóricos más tradicionales. Aunque de por si el artículo logró que el modelo propuesto explique bien los eventos complejos reportados por la crisis.

Por último, tomando en cuenta las debilidades encontradas para un próximo avances en la pregunta propuesta, el autor puede incluir una sección en la que compare estos modelos que mostró que aún pueden ser aplicados en contextos contemporáneos, con algunos modelos modernos que proponen otros investigadores, de manera que se obtiene un mejor panorama para comprender estas crisis.

Bibliografía:

Mendoza, W. (2015). Demanda y oferta agregada en presencia de políticas monetarias no convencionales

A partir del siguiente sistema de ecuaciones que representan el modelo IS-LM#

1. Encuentro con las ecuaciones de Ingreso (\( Y^e \)) y tasa de interés (\( r^e \)) de equilibrio (Escriba paso a paso la derivacion de estas ecuaciones).#

Por un lado, necesitamos considerar la ecuación de la curva IS

\[ r = \frac{B_o}{h} - \frac{B_1}{h}Y \]

  • En la cual \( B_0 = C_o + I_o + G_o + X_o \) y \( B_1 = 1 - (b - m)(1 - t) \)

Por otro lado, tenemos la ecuación de la curva LM:

\[ r = -\frac{1}{j}\frac{Mo^s}{P_o} + \frac{k}{j}Y \]

Al igual ambas ecuaciones, podemos encontrar el nivel de Ingresos equilibrio \((Y^e)\) y la tasa de interés de equilibrio \((r^e)\):

\[ -\frac{1}{j}\frac{Mo^s}{P_o} + \frac{k}{j}Y = \frac{B_o}{h} - \frac{B_1}{h}Y \]

  • Ingreso de equilibrio:

\[ Y^e = \frac{j B_o}{k h + j B_1} + (\frac{h}{k h + j B_1})\frac{Ms_o}{P_o} \]

  • Tasa de interés de equilibrio:

\[ r^e = \frac{kB_o}{kh + jB_1} - (\frac{B_1}{kh + jB_1})\frac{Ms_o}{P_o} \]

Estas dos ecuaciones representan el modelo IS-LM y los puntos o valores de equilibrio

2. Grafique el equilibrio simultáneo en los mercados de bienes y de dinero.#

import matplotlib.pyplot as plt
import numpy as np
import sympy as sy
import pandas as pd
from sympy import *

# Curva IS

# Parámetros

Y_size = 100 

Co = 35
Io = 40
Go = 50
Xo = 2
h = 0.8
b = 0.4
m = 0.5
t = 0.8

Y = np.arange(Y_size)


# Ecuación 
def r_IS(b, m, t, Co, Io, Go, Xo, h, Y):
    r_IS = (Co + Io + Go + Xo)/h - ( ( 1-(b-m)*(1-t) ) / h)*Y  
    return r_IS

r_is = r_IS(b, m, t, Co, Io, Go, Xo, h, Y)


#--------------------------------------------------
    # Curva LM 

# Parámetros

Y_size = 100

k = 2
j = 1                
Ms = 200             
P  = 20               

Y = np.arange(Y_size)

# Ecuación

def r_LM(k, j, Ms, P, Y):
    r_LM = - (1/j)*(Ms/P) + (k/j)*Y
    return r_LM

r_lm = r_LM( k, j, Ms, P, Y)

# Gráfico del modelo IS-LM

# Dimensiones del gráfico
y_max = np.max(r_lm)
fig, ax = plt.subplots(figsize=(10, 8))

# Curvas a graficar
# Curva IS
ax.plot(Y, r_is, label = "IS", color = "green") #IS
# Curva LM
ax.plot(Y, r_lm, label="LM", color = "red")  #LM

# Eliminar las cantidades de los ejes
ax.yaxis.set_major_locator(plt.NullLocator())   
ax.xaxis.set_major_locator(plt.NullLocator())

# Texto y figuras agregadas
# Graficar la linea horizontal - r
plt.axvline(x=51.5,  ymin= 0, ymax= 0.52, linestyle = ":", color = "black")

# Plotear los textos

plt.text(53,-10, '$Y_0$', fontsize = 12, color = 'black')

# Título, ejes y leyenda
ax.set(title="Equilibrio simultáneo en los mercados de bienes y de dinero", xlabel= r'Y', ylabel= r'r')
ax.legend()

plt.show()

Código (Tres puntos Extra!)#

A partir del siguiente sistema de ecuaciones que representan el modelo IS-LM#

1. Encuentre con las ecuaciones de Ingreso (\( Y^e \)) y tasa de interés (\( r^e \)) de equilibrio (Escriba paso a paso la derivacion de estas ecuaciones).#

Teniendo la tasa de interes nominal donde: $\( i = r+π^e \)$

  • Mercado de Bienes IS $\( [1-(b-m)(1-t)]Y - (C_0+G_0+x_0) = I_0 - h(r+π^e) \)$

\[ hi = [I_0+C_0+G_0+x_0]-[1-(b-m)(1-t)] \]
\[ i= \frac{1}{h} (I_0+C_0+G_0+x_0) - [\frac{1-(b-m)(1-t)}{h} ] Y\]

Donde: \( B_0 \) es \( I_0+C_0+G_0+x_0 \) y \( B_1\) es \( 1-(b-m)(1-t) \)

  • Mercado de dinero LM $\( M^d = M^s \)$

\[ kY-j(r+π^e) = \frac{Mo^s}{P_o} \]
\[ kY-\frac{Mo^s}{P_o} =ji \]
\[ kY-\frac{Mo^s}{P_o} =ji \]
\[ \frac{kY}{j}-\frac{Mo^s}{P_o} = i \]

\( Obtenemos: \)

\[ i = -\frac{Mo^s}{P_o}+ \frac{kY}{j}\]

  • Al igualar ambas ecuaciones tenemos: $\( -\frac{1}{j}\frac{Mo^s}{P_o} + \frac{k}{j}Y = \frac{B_o}{h} - \frac{B_1}{h}Y \)$

\(Finalmente: \)

  • Ingreso de equilibrio:

\[ Y^e = \frac{j B_o}{k h + j B_1} + (\frac{h}{k h + j B_1})\frac{Ms_o}{P_o} \]

  • Tasa de interés de equilibrio:

\[ r^e = \frac{kB_o}{kh + jB_1} - (\frac{B_1}{kh + jB_1})\frac{Ms_o}{P_o} \]
